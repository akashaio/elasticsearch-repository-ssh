Elasticsearch Repository SSH
=======================

## Overview

Repository SSH Plugin provides SSH repository for Elasticsearch's Snapshot/Restore feature.

## Version

| Version   | Elasticsearch |
|:---------:|:-------------:|
| master    | 1.4.X         |
